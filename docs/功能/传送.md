# 传送

跨越千里，如何移动？

## 命令列表

| 命令         | 说明             |
|------------|----------------|
| /tpa       | 请求将阁下传送到一个玩家   |
| /tpahere   | 请求把一个玩家传送到阁下身边 |
| /tpaccept  | 接受一个传送请求       |
| /tpdeny    | 拒绝一个传送请求       |
| /tpignore  | 切换是否忽略传送请求     |
| /tpdecline | 拒绝一个传送请求       |
| /back      | 回到上一个位置        |
| /tpr       | 随机传送到世界某个位置    |

## 功能说明

使用如上命令可以执行传送操作。

!!! tip

    若阁下正在使用 《Minecraft: Java版》 进行游玩，可以直接通过点击聊天栏中的选项来处理传送请求。
