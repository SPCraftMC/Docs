## 命令列表

| 命令         | 说明             |
|------------|----------------|
| /tpa       | 请求将阁下传送到一个玩家   |
| /tpahere   | 请求把一个玩家传送到阁下身边 |
| /tpaccept  | 接受一个传送请求       |
| /tpdeny    | 拒绝一个传送请求       |
| /tpignore  | 切换是否忽略传送请求     |
| /tpdecline | 拒绝一个传送请求       |
| /back      | 回到上一个位置        |
