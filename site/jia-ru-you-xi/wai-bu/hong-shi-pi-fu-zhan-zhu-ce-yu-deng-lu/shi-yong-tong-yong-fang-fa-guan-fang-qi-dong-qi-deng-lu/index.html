    使用通用方法登录

    警告：这将使所有验证转向我们指定的验证服务器（红石皮肤站）

    前往https://authlib-injector.yushi.moe/下载authlib-injector

    假设：

    • 你下载到的 authlib-injector JAR 文件名为authlib-injector.jar
    • 你将其放到了C:/
    • 你的客户端 JAR 名称为client.jar

    打开你的启动器，修改启动参数，在java 命令后方添加-javaagent:C:/authlib-injector.jar={验证服务器地址}

    // 注意：只是示例，切勿直接复制粘贴下方启动参数，否则将无法启动
java -javaagent:C:/authlib-injector.jar={验证服务器地址} ... -jar client.jar
